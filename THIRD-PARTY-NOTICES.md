# Third-Party Notices

Very Good Document Imposer is licensed under 0BSD (see `LICENSE`). 0BSD waives attribution for
*our* code, but does not discharge the inbound obligations of the permissively-licensed
dependencies we statically link and redistribute (SPEC §3.1). This file enumerates those.

The Rust dependency-graph section is generated in CI by `cargo about generate` (and the
allowed-license set is enforced by `cargo deny check licenses`). The native section below is
hand-maintained because the qpdf vendored bundle is compiled from source, not pulled as a crate
with machine-readable metadata.
