# Very Good Document Imposer

An open-source implementation of all major document imposition workflows on a platform-agnostic Rust-based core.

VGDI is made up of several parts:

- The core imposition engine (VGDI)
- The CLI tool (VGDI-CLI)
- Three bespoke UI layers, native on each platform (Windows, Mac, Linux)

Currently, this project is in early alpha and is actively being worked on.
